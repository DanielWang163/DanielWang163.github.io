HTML note 3

<img>

  • Defines an image in a HTML page

src

  • <img> Attribute
  • Specifies the URL of an image
  • There are 2 different ways to use this attribute
    • 1. Using URL of Image
    • 2. Saving the Image File
  • Step for method 2: The image should be saved in the folder containing the HTML code

height & width ( <img> Attribute )

  • <img> Attribute
  • Used to resize the image in pixels
  • You can specify only one height or width and the aspect ratio will be adjusted
