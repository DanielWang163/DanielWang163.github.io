text-allign

  • CSS Propety
  • Specifies the horizontal alignment of text
  • Left: Aligns the text to the left
  • Right: Aligns the text to the right
  • Center: Centers the text

text-decoration

  • CSS Property
  • Adds an underline, over-line, line-through, to selected text
  • None: No text-decoration
  • Underline: Horizontal line underneath text
  • Overline: Horizontal line above text
  • Line-through: Strikethrough. Horizontal line through the middle of text

font-weight

  • CSS Property
  • Bold or light text
  • Normal: Standard weight. Equivalent of 400
  • Bold: Bold weight. Equivalent of 700
  • 100: Lightest
  • 900: Boldest

font-family

  • CSS Property
  • Specifies the font for an element
  • Serif: Small line attached to the end of a stroke in a letter
  • Sans-Sherif: Does not have extending features at the end of strokes
  • Monospace: Fixed-width font (Code font)
  • Cursive: Flowing font
  • Fantasy: Daft font

<span>

  • Body element
  • Used for grouping and applying styles to inline elements

id

  • Attirbute
  • Specifies a unique id for an HTML element
  • Can be used on any HTML element
  • When using an ID, the selector should start with `#`
