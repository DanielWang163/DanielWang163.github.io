CSS Notes 4

border-style

  • CSS Propety
  • Sets a style for the border
  • Values:
    • none
    • dotted
    • dashed
    • solid
    • double
    • groove
    • ridge
    • inset

border-width

  • CSS Property
  • Sets a width for the border
  • Values:
    • medium
    • thin
    • thick
    • _px

border-color

  • CSS Property
  • Sets a color for the border

padding

  • CSS Property
  • Used to give space inside the border
  • Properties
    • padding-top
    • padding-right
    • padding-bottom
    • padding-left

Margin

  • CSS Property
  • Used to give space outside the border
  • Properties:
    • margin-top
    • margin-right
    • margin-bottom
    • margin-left
